# Contributing

Thanks for considering a contribution.

## Scope

Preferred contribution areas:

- Backup reliability
- Export quality
- Folder and dashboard UX
- Documentation
- Bug fixes related to Gemini page structure changes

Out of scope for the public OSS build:

- Reintroducing excluded watermark-removal behavior into the main extension build

## How to contribute

1. Fork the repository.
2. Create a small focused branch.
3. Describe the problem and the user impact.
4. Keep changes local-first and privacy-conscious.
5. Open a pull request with testing notes.
