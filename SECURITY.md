# Security

If you believe you found a security issue in this repository, please avoid posting sensitive exploit details in a public issue.

Preferred first contact:

- GitHub issue with limited reproduction details for low-risk bugs
- Direct maintainer contact for higher-risk findings

When reporting an issue, please include:

- A short description of the impact
- Steps to reproduce
- Screenshots or logs when safe to share
- Whether the issue affects backup privacy, local storage, or export behavior
