# Open Source Scope

This directory is the public OSS-safe package for GeminiSaver Fairy.

## Included in the public build

- Conversation backup
- TXT and Markdown export
- Backup dashboard
- Folder organization
- Gem icon customization
- Prompt clipboard helpers

## Excluded from the public build

- Experimental watermark-removal behavior

The excluded files were moved out of the runtime path and are ignored by Git through `.gitignore` so the public repository can stay focused on transparent, maintainable, lower-risk functionality.
